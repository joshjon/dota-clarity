# Get Favourite Match

Get a match that a Dota Clarity user has favourited.

**URL** : `/matches/favourites/{match_id}`

**Method** : `GET`

**URL Params** :

Required

| Attribute | Type    | Description                       |
| :-------- | :------ | :-------------------------------- |
| id        | string  | user id of a dota clarity account |
| match_id  | integer | dota 2 match id                   |

## Success Responses

**Code** : `200 OK`
